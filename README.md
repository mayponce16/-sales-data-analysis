# 📊 Sales Data Analysis Project

## 🧠 Project Overview
This project analyzes sales data to identify key business insights such as top-performing cities, products, and customers.

---

## ⚙️ Tools Used
- Python
- Pandas
- Jupyter Notebook

---

## 🔍 Process

### Data Exploration
- Used `head()`, `info()`, `describe()`

### Data Cleaning
- Handled missing values
- Standardized text data

### Data Analysis
- Total sales calculation
- Grouping by city, product, and customer

---

## 📈 Key Insights

- CDMX generates the highest revenue  
- Laptop is the most profitable product  
- Ana is the highest-value customer  

---

## 🚀 Conclusion

The analysis highlights key opportunities to focus on high-performing products and locations.
